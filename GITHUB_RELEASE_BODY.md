# Home Assistant Multi-Zone Heating v1.1.0

A generic, privacy-safe release for scheduled room heating, automatic away setback, three-mode presence control, safety interlocks and a responsive standalone dashboard.

## Highlights

* Generic dummy entities throughout
* Only heating-relevant YAML
* AI customization prompt and entity worksheet
* Manual setup guide
* Panel, floor and multi-heater examples
* Desktop and mobile dashboard screenshots

Start with `README.md`, then complete `docs/ENTITY_WORKSHEET.md`.
